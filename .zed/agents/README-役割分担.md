# エージェントの役割分担（1エージェント1作業）

各エージェントは単一責務のみを担当し、1回の呼び出しで複数作業を実施しない。

## 役割一覧

| 役割 | 担当 | 作業 |
|------|------|------|
| UIデザイン | `ui-designer` | 情報設計・優先順位・運用を重視したUIを設計する |
| 事前調査 | `frontend-context` | フロントの依存関係と競合を調査する |
| 退行検証 | `regression-verifier` | 変更起因の退行を判定する |
| Web検索・最新情報 | `web-researcher` | Web検索で最新情報を取得し報告する |

## 呼び出し順の基本形

1. 変更直後の確認は `regression-verifier`
