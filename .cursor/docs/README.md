# docs

調査メモ・設計メモ・作業記録など、**知識（`.cursor/order/`）以外**の Markdown を置く。

| 置き場所 | 内容 |
|----------|------|
| `.cursor/order/`（プロジェクト） | 技術スタック・機能仕様・作業手順（SSOT） |
| `.cursor/docs/`（プロジェクト） | セッション単位の記録・調査結果 |

作成: グローバルコマンド `/create-doc-md`（`~/.cursor/commands/`）

グローバル設定（commands / agents / skills）は `~/.cursor/` を参照。詳細は `.cursor/README.md`。
